# Documentação de Tarefa

Ao **finalizar** uma tarefa, documente a entrega:

- Resumo e descrição simples
- Evidência de validação
- O que foi corrigido
- Comentário para ticket

Use `/documentacao-tarefa` para gerar o texto pronto para colar.
